## Running the Simulated Annealing Monte Carlo Sampling Pipeline

Before running the simulated annealing pipeline, make sure you have already completed the **TAD** and **compartment score** pipelines.

The simulated annealing workflow consists of the following steps:

### Step 1: Assign Cell Types

Run:

```text
scripts/ami_3celltypes.py
```

Second, merge the output file from this script with the pre-computed energy values generated by `scripts/Ising-Energy/compute_energy.py`. Alternatively, you can use the pre-generated file included in the repository:

```text
data/non_neuron_filtered_celltypes.tsv
```

### Step 2: Generate Random Barcodes

Run:

```text
scripts/simulated-annealing-monte-carlo/generate_barcodes.py
```

This script generates a barcode file that will be used as the input for the simulated annealing algorithm.

### Step 3: Run the Simulated Annealing Algorithm

Use the barcode file generated in **Step 2** as the input for:

```text
scripts/simulated-annealing-monte-carlo/sa_desktop.py
```

This script performs the simulated annealing Monte Carlo optimization and generates the output files required for the downstream analyses and figures.

> **Note:** Each script contains a dedicated `README.md` with detailed instructions, descriptions of the input and output files, and configurable parameters. Please refer to the corresponding README before running each step.
