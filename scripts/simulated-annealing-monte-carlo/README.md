## Generate Metadata file

### Step 1: Generate the AMI similarity matrix

First, obtain the AMI similarity matrix by running:

```bash
scripts/TAD-detection/2.1_scKTLD_TADs_detection.py
```

This script generates several output files. For the downstream analysis described here, use the file:

```
simi_ami.xlsx
```

### Step 2: Compute the 2D MDS embedding

Next, run:

```bash
ami_3celltypes.py
```

This script:

- loads the `simi_ami.xlsx` similarity matrix,
- filters the cells to the selected cell types (ASC, OGC, and OPC),
- computes a two-dimensional embedding using **Multidimensional Scaling (MDS)**, and
- exports the MDS coordinates together with the corresponding cell type information.

The output file is:

```
ami_mds_coordinates_ASC_OGC_OPC.tsv
```

which has the following format:

| barcode | mds_component_1 | mds_component_2 | celltype |
|----------|----------------:|----------------:|----------|
| AAACCTGAGT | -0.42 | 1.18 | ASC |
| ... | ... | ... | ... |

This table is used as the input for the subsequent analysis steps.

## Step 3: Merge the MDS coordinates with the Ising energy

Once the MDS coordinates have been generated, merge the file

```text
ami_mds_coordinates_ASC_OGC_OPC.tsv
```

with the Ising energy table

```text
new_energy.tsv
```

generated by:

```bash
scripts/Ising-Energy/compute_energy.py
```

Merge the two files using the `barcode` column. The resulting table combines the MDS coordinates, cell type annotation, Ising energy, and cell category for each cell.

The merged file should have the following format:

| barcode          |     mds_component_1 |     mds_component_2 | celltype |              energy | category   |
| ---------------- | ------------------: | ------------------: | -------- | ------------------: | ---------- |
| AAACGAAGTGACAAGC | -0.0467182724060481 | -0.3498509534467920 | OGC      | -0.7887303775333190 | Non-neuron |
| AAACGAATCGGATGTT | -0.0409951834853905 | -0.2559396887995190 | OGC      |  -0.716188458056023 | Non-neuron |

This merged table serves as the **metadata file** for all subsequent analyses in this repository.

For convenience, a ready-to-use example of the merged metadata file, `non_neuron_filtered_celltypes.tsv`, is provided in the `data/` directory.
